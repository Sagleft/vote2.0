# Vote 2.0

Тема сервиса на форуме проекта Freeland: https://forum.mfcoin.net/topic/492/

## Требования
* PHP 7.1;
* MySQL \ MariaDB;
* Apache 2.6;

## Развертывание

```
git clone https://github.com/Sagleft/vote2.0.git vote2
cd vote2
mkdir view/cache
composer update
```

Directory: ``` controller\public_html ```
